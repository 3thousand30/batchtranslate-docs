# BatchTranslate with AI - Documentation

Official documentation for **BatchTranslate with AI**, a Windows desktop app by [3thousand30](https://3thousand30.com) for translating folders of PDF, DOCX, TXT, and Markdown files into one or more languages using your own AI provider key.

## What is BatchTranslate with AI?

BatchTranslate with AI helps you translate a whole folder of documents from a Windows desktop app you control. You bring your own AI key, choose the provider and model that fit your budget, and the translated files are written back to your PC as Markdown or plain text.

## Key features

- **Batch translation** for folders of PDF, DOCX, TXT, and Markdown files
- **Multiple target languages** in a single run
- **Long-document handling** with chunked translation for books and large files
- **Auto-detect source language** with optional manual override
- **Prompt-caching savings** on supported providers for multi-language runs
- **Scanned PDF detection** so image-only PDFs are skipped with a clear reason
- **Local-first workflow** with encrypted API key storage via Windows DPAPI

## Privacy

BatchTranslate with AI does not store your documents or translated outputs on 3thousand30 servers. You connect directly to your chosen AI provider using your own API key. See the [privacy policy](https://3thousand30.github.io/batchtranslate-docs/privacy.html) for details.
